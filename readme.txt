B-east for WooCommerce is a WordPress plugin to add the B-east shipping option to your WooCommerce shop.

== Description ==

With this extension, you can send your WooCommerce shop orders directly to B-east. Your shipping labels will be available in WooCommerce's back end!

== Installation ==

= Manual installation =
1. Download the plugin
2. Go to WordPress admin and go to Plugins > Add New Plugin
3. Select Upload Plugin
4. Choose the zip file
5. Activate the plugin!

= Setting up the plugin =
1. Go to WooCommerce > B-east
2. Enter your B-east API key

== Frequently Asked Questions ==

= How do I get a B-east API key? =

Contact B-east customer support at info@b-east.nl.

== 3rd Party Services ==

Our plugin connects to third-party services. We use WooCommerce order data to create a shipping label via B-east's API. More information can be found at https://app.b-east.nl/privacy-policy.
